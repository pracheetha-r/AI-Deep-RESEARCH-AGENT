# AI Deep RESEARCH AGENT
This project is an AI-powered Research Assistant that lets users enter any research topic through a Streamlit web interface. The request is sent to a FastAPI backend, where a Research Agent orchestrates the entire workflow. The agent uses a Search Tool to gather real-time information from the web and a Groq LLM to analyze, summarize, and structure the findings into meaningful insights. The final response is returned as clean, structured JSON and displayed back to the user through the interface, creating a complete end-to-end automated research pipeline.
